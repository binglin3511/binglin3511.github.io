react-native准备工作

###准备工作-安装

####1、Homebrew&&NodeJS

react-native依赖于NodeJS，可以通过Homebrew安装。MAC环境下brew默认已安装，Homebrew安装命令行如下：

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

安装NodeJs，react native依赖NodeJS版本4.0以上，默认Homebrew安装的NodeJS版本为6.0（但是亲测安装NodeJS版本为v4.2.3）

brew install node

####2、react native命令行工具
react native命令行工具能够方便快捷的新建项目

npm install -g react-native-cli
